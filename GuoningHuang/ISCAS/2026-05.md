### 本月工作总结

本月主要围绕 buddy-mlir 项目中的 GPU 推理优化与 JIT 训练两个方向展开工作。在 GPU 推理方向，为 DeepSeek-R1-Distill-Qwen-1.5B 实现了完整的端到端 GPU 推理流程，新增 matmul GPU tiling、嵌套 parallel loop、strided copy 并行化等优化 pass，并实现了基于 CUDA managed memory 的自定义运行时库，在 A100 上实测 prefill 约 27s（1024 tokens），decode 约 0.96s/token；在 JIT 训练方向，以两层 MLP 为例实现了 PyTorch + Buddy MLIR JIT 的训练路径，支持仅编译前向传播或同时编译前向与反向传播两种模式，三种运行模式（纯 PyTorch、JIT 前向、JIT 前向+反向）数值结果完全一致。

### 本月提交的PR

- https://github.com/buddy-compiler/buddy-mlir/pull/794 为 `examples/BuddyDeepSeekR1-GPU` 新增完整 GPU 推理 pipeline（`[examples] Add DeepSeekR1 GPU inference with optimization passes`），包含 GPU matmul tiling/shared memory/GEMV 优化 pass、CUDA managed memory 运行时库，以及 prefill 与 decode 的 GPU kernel 编译流程。

- https://github.com/buddy-compiler/buddy-mlir/pull/792 实现 PyTorch + Buddy MLIR JIT 训练路径（`[examples][frontend] Add BuddyJITTrain: MLP training via Buddy MLIR JIT`），支持前向/反向传播编译，修复 DynamoCompiler 中参数排序、梯度重建等若干问题。
